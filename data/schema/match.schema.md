# Match Schema

Represents a football match.

Fields

id
league_id
season
matchday
home_team_id
away_team_id
kickoff
status

Result

home_goals
away_goals

Example

id: bundesliga-2025-bayern-dortmund
league_id: bundesliga
season: 2025
matchday: 12

home_team_id: bayern
away_team_id: dortmund

kickoff: 2025-11-08T18:30
status: scheduled
