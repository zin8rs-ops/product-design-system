# Prediction Schema

Represents user or AI match predictions.

Fields

match_id
user_id
prediction_type
predicted_home_goals
predicted_away_goals
confidence
created_at

Example

match_id: bayern-dortmund-2025
user_id: user_123

prediction_type: score

predicted_home_goals: 2
predicted_away_goals: 1

confidence: 0.74
