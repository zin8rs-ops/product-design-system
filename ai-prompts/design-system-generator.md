# Design System Generator

This prompt instructs AI tools to generate pages using the football design system.

---

## Design System Structure

tokens → design values  
components → UI building blocks  
patterns → reusable sections  
templates → full page structures  
examples → visual references  

---

## Generation Rules

1 Use templates to determine the page structure.

2 Use patterns to build page sections.

3 Use components to implement UI elements.

4 Apply tokens for colors, spacing, typography, radius and shadows.

5 Do not invent new components unless absolutely necessary.

---

## Example: Generate a Competition Page

Template

league-page

Patterns

match-list

---

## Example Prompt

Generate a Serie A competition page using the design system.

The page should include:

- competition hero
- match list
- league table
- top players
- highlights
- news feed

Use only components and patterns defined in the design system.
