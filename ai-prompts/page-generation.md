# Page Generation Rules

Generate UI pages using the design system.

## Steps

1 Use templates to determine page structure  
2 Use patterns for sections  
3 Use components for UI elements  
4 Apply design tokens for styling  

## Example

World Cup Page

Template → world-cup-page  
Patterns → tournament-hero, match-list, group-table-grid, stats-section  
Components → match-card, team-row, player-row  

## Rules

- Use tokens for all spacing, colors and typography
- Do not invent new components
- Prefer existing patterns
